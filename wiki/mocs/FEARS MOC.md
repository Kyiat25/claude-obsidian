# FEARS — Map of Content

> A Map of Content (MOC) links into a cluster of related atomic notes. It is a navigation hub, not a container. Notes are not stored under this MOC; they live flat under `wiki/notes/` and are reached by following these links.

## Why this MOC exists

FEARS is a tabletop RPG system (backwards-compatible with 5E) introduced by the source document [[Welcome to FEARS]]. This MOC is the entry point into everything the wiki knows about its mechanics and character archetypes — it exists because FEARS is the first content in an entirely new domain for this vault (everything else here is about the claude-obsidian plugin itself).

### Class roster — 24 classes, all with dedicated pages
Full table with caster tier, resource, and hit die: [[FEARS Class Roster]]. Per-class deep-dive files in `.raw/` are now being ingested one at a time, upgrading each class's existing page in place (same in-place-upgrade pattern used throughout this cluster) rather than creating duplicates. **Standard as of the Bard ingest**: class pages carry every named ability in full mechanical detail (full text, not condensed bullets), not just a curated subset — [[Barbarian]] and [[Bard]] were both retroactively expanded to this standard per explicit user direction. **Progress: ALL 24/24 classes done.** [[Barbarian]], [[Bard]], [[Apothecary]], [[Bender]], [[Binder]], [[Cleric]], [[Druid]], [[Fighter]], [[Inventor]], [[Investigator]], [[Magus]], [[Monk]], [[Paladin]], [[Ranger]], [[Reaper]], [[Rogue]], [[Sangromancer]], [[Shaman]], [[Sorcerer]], [[Tattooist]], [[Warden]], [[Warlock]], [[Witch]], [[Wizard]] — the per-class deep-dive pass that began with Barbarian is now complete. Every class in the FEARS roster has a fully-detailed page: full level progression, every named ability in full mechanical text (not condensed bullets, per the standard set after the Barbarian/Bard retroactive-expansion pass), Last Stand options, and Class Reactions. Two universal per-class mechanics confirmed so far: [[Last Stand]] (permanent-death capstone, 3 options — though [[Bender]]'s Elemental Rebirth is a confirmed exception to the death clause; every other class so far reaffirms the standard clause) and [[Class Reactions]] (PB-sized shared reaction pool, called "Reaction Points" per [[Bender]] and [[Binder]]) — both spun into dedicated hub pages, now at 15/24 each. Class Reactions' pool exceptions keep deepening (Cleric's/Investigator's single-reaction exceptions, Druid's independent-per-reaction budgets, Inventor's 7th-level gate, Magus's and now [[Reaper]]'s heavy secondary-resource layering, Monk's total Focus swap), but [[Paladin]] and [[Ranger]] both gave clean, exception-free confirmations right before it. The `8+PB+ability mod` DC formula is now confirmed eleven times, spanning WIS/CHA/INT casters and two non-casters ([[Investigator]], [[Monk]]) — clearly a universal DC shape for this game. **Unresolved pattern**: Last Stand save DCs now split three ways — `10+PB+ability` (Cleric, Apothecary, Inventor), `12+PB+ability` (Bender, Monk, Paladin×2), and plain `8+PB+ability` (all three of [[Reaper]]'s options), while [[Ranger]]'s options state no DC at all. Damage Reduction and Evasion have now each accumulated a third DCS-summary-vs-deep-dive contradiction ([[Ranger]]'s Hunter's Mark DR and Favoured Enemy Evasion) — a recurring pattern worth naming explicitly: the original DCS-level summaries seem to have been loose approximations in several places, not just isolated one-offs. [[Reaper]]'s deep-dive is by far the largest and most complete of the cluster (1237 lines, three fully detailed Covenant subclasses) — a strong signal that at least some per-class files carry full subclass crunch this generic document format hasn't shown elsewhere. - **Minor documentation quirk**: [[Warlock]] reuses the feature name "Eldritch Resilience" for two unrelated ability suites (9th-level Pact Shield/Woven Fate/Supernatural Resolve, and a 15th-level Injury downgrade) — the same kind of internal naming inconsistency flagged on [[Reaper]], [[Shaman]], and [[Paladin]] this session.
- **New, deliberately left open (not a contradiction)**: [[Witch]]'s per-class deep-dive never mentions "Hex of Frailty," the ability [[Damage Reduction]] currently credits to Witch as a "zero drift" DCS-level hook. Unlike this session's several outright DR contradictions (Barbarian/Monk/Paladin/Ranger/Shaman), this deep-dive also never details Coven-specific features beyond two named examples — so the ability may simply live in undetailed Coven content rather than being contradicted. Flagged as unconfirmed, not resolved either way.
