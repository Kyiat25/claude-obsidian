# Memory

One of three knowledge skills (with [[Occult]] and [[Streetwise]]). Lets a character reconstruct lore, rituals, and past relationships; avoid missteps with previously-encountered NPCs or regions; resist false memories and echo-based illusions.

## Formula and use (from a dedicated `Memory.md` player-guide source)

**Memory Check** = `1d20 + INT modifier + Proficiency Bonus (if proficient)`, granted by classes like Investigator/Theurge/Thaumaturge, memory/trauma-themed backgrounds, or glyph-binding training. Typical DCs: recalling a glyph seen in a vision (12-18), rebuilding forgotten lore or ritual (15-25, varies by depth), resisting memory alteration (contested vs. the spell's DC). Explicitly *not* [[Sanity]] — Memory measures cognitive clarity, not stability, though low Sanity imposes disadvantage on trauma-linked Memory checks and high [[Corruption]] can seed false memories. The [[Whisper Engine]] may require a Memory check before spending Whisper Points, to reconstruct or decode a Whispered glyph. Proficiency is trained via 4 downtime days of focused study, tutelage under an Echo-Keeper/Psion/Glyph-Warden, or surviving a full Eldritch Vision with a successful Sanity Save (DC 18).
