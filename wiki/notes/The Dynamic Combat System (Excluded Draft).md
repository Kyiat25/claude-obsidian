# The Dynamic Combat System (Excluded Draft)

**This is the file every prior FEARS ingest flagged as "relationship to DCS.md not yet checked."** The answer: it's an earlier, larger draft — subtitled "Version 2.0" — that the source material's own frontmatter marks `status: excluded`. Unlike every other DCS-companion source ingested so far (Combat Flow, Called Shot, Damage Reduction, Momentum, Evasion, Injury System — all of which confirmed the current [[Dynamic Combat System (DCS)]] with zero drift), **this one does not confirm current canon — it predates and was superseded by it.**

## How this ingest was handled

Nothing in this document was used to update any confirmed page's numbers. Where it differs from current canon, that's treated as **design history** (what changed), not as a contradiction to resolve. Where it covers ground no confirmed source has reached yet (Initiative, monster conversion, DM guidance), it's presented as an **unverified preview**, clearly caveated, not added to those topics' eventual canonical pages.

## Confirmed deltas (this draft vs. current DCS)

- **Momentum range**: this draft used **−3 to +3** (7 states: Dominating/Flowing/Confident/Neutral/Pressured/Struggling/**Overwhelmed**). Current [[Momentum]] uses **−2 to +3** (6 states) — the −3 "Overwhelmed" tier (first attack disadvantage, attacks against you have advantage) was cut entirely.
- **Class-specific Momentum rules differ substantially.** This draft's Barbarian ("first critical hit per rage grants +3 momentum instead of +2"), Monk ("spend 1 Ki to maintain momentum at +3"), Rogue ("Sneak Attack grants +2 momentum, never suffer disadvantage at −3"), and Fighter ("Second Wind resets momentum to +1") rules do not match any confirmed [[Barbarian]], [[Monk]], [[Rogue]], or [[Fighter]] page content — all were apparently reworked or removed.
- **Called Shot DR-bypass framing is numerically consistent** with confirmed [[Damage Reduction]] (Light 5/Medium 10/Heavy 15 Total DR; same per-location breakdown) — the armor math survived the revision unchanged.
- **Called Shot save DC formula**: `8 + Proficiency Bonus + attack ability modifier`. No other confirmed source states this number. **Update — adopted by user ruling**: promoted to confirmed canon on [[Called Shot]] despite this source's excluded status, since it's the only candidate that exists, matches the house `8+PB+ability` DC pattern used everywhere else, and the ruling judged that adopting it doesn't disturb Called Shot's actual risk/reward balance (which runs through the attack-roll penalty, not this save). A narrow, source-specific promotion — the rest of this draft's deltas (Momentum range, class-specific rules) remain design-history only, not adopted.
- **Called Shot subclass reductions differ**: this draft's Ranger gets a flat "−1 to all called shot penalties," while confirmed [[Called Shot Subclass Features]] gives Hunter's Mark a non-uniform −2/−1/−1 reduction. Reworked, not contradictory (different draft stages).

## Confirmed preview: Initiative

**Update — verified accurate.** `Initiative.md` has since been ingested (see [[Initiative]]), and the core two-phase system this draft previewed in Chapter 7 (traditional `1d20+DEX` in round 1, Momentum-average-driven from round 2, Focus-based leadership tiebreaker, routed creatures excluded from the average) survived into confirmed canon essentially unchanged. This is the **first piece of this excluded draft to be independently confirmed rather than superseded** — meaningful, since everything else checked so far (Momentum range, class-specific rules, Called Shot DC) was reworked or cut. Not everything in an excluded draft is necessarily wrong; some of it just hadn't been formally re-confirmed yet. The confirmed source adds an entire mechanic this draft never mentioned at all: [[Seize the Moment]], a universal PC reaction with no precedent here.

## Unverified previews (no confirmed source covers these yet)

- **Monster conversion guidelines** (Chapter 11): a by-size DR/Momentum-range table (Small/Medium/Large/Huge/Gargantuan) and worked conversions (Goblin, Troll, Adult Red Dragon) for adapting standard 5E stat blocks into DCS. Not confirmed anywhere else.
- **DM implementation guidance** (Chapters 10, 8-9): a phased rollout plan for introducing DCS to a table (Foundation → Rhythm → Precision → Full System over ~10 sessions), player-facing tactical guidelines, and character-creation advice. Purely pedagogical/design-philosophy content — likely still relevant regardless of the mechanical excluded status, but not verified against anything else in this cluster.

## Why flag this at all if it's excluded?

Because it's the first source in this entire cluster that documents **why** something changed, not just what the current rule is. Every other contradiction in this wiki ([[Psion]]/[[Runesmith]]/etc. retirements, the Focus/Technique-Point naming conflict, the Injury System scope error) was inferred by comparing sources against each other. This one is explicit: the design team tried a version, then excluded it. That's a different, more valuable kind of signal — worth preserving as the cluster's only direct evidence of an iteration cycle.
