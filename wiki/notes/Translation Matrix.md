# Translation Matrix

The mechanism by which FEARS claims **total backwards compatibility** with existing 5E characters, modules, and campaigns — "what changes is not the core of your story, but how your story unfolds."

## Named components

- **Attribute Mapping** — "direct translation of core attributes" (no mapping table given in this source)
- **Skill Transformation** — enhances skill interactions; introduces new **[[Focus]]** and **[[Sanity]]** attributes plus **[[Streetwise]]** skills for "more intense narrative moments" — all three defined in [[A World that Watches]], not here
- **Spell and Ability Conversion** — existing spells/abilities keep their functionality but gain "dynamic execution" under the new rules (e.g. the [[FEARS Combat System]] and [[Unstable Magic Zones]])

## What's not specified

No full attribute-to-attribute table exists yet, but [[FEARS Character Framework]] gives the first concrete conversion steps: track Evasion via `1d20 + DEX + PROF` instead of AC, replace armor's AC bonus with per-body-part Damage Reduction, add Whisper Points via `3 + Focus Mod + ½ Sanity Mod`, and begin tracking Corruption Score / [[Mental State]] / Reaction options / Called Shot effects. Still missing: the actual Sanity and Focus *scores* (how a 5E character gets starting values for attributes 5E never had).
