# FEARS Advanced Backgrounds

A 4-rank progression system that turns a PHB-style background from static character history into an ongoing advancement track running alongside class levels. **The system is now complete**: 61 backgrounds across all 3 batches, matching Batch 3's own closing count exactly. Each background grants, at Rank 1: skill/tool/language proficiencies, starting equipment, a named Feature, and starting Holdings. Each subsequent rank (2, 3, 4) grants expanded Holdings, an upgraded/expanded Feature, and requires completing a specific in-fiction Progression trigger to unlock — there is no XP or session-count gate alone; the trigger is the gate.

## Power budget per rank

- **Rank 1**: 1-2 points (skill, tool, equipment, minor feature)
- **Rank 2**: 2-3 points (better holdings, expanded feature)
- **Rank 3**: 3-4 points (leadership, significant influence)
- **Rank 4**: 4-5 points (mastery, major impact)

No formula is given for what a "point" converts to mechanically — this reads as a design-time balancing heuristic for the person writing new backgrounds, not a player-facing resource to spend.

## Progression timing

> [!contradiction] Batch 1 is the outlier — Batches 2 and 3 agree with each other
> Batch 1: Rank 1→2 = 3-5 sessions, Rank 2→3 = 5-8 sessions, Rank 3→4 = 8-12 sessions.
> Batch 2's "Progression Summary" and Batch 3's "Experience Requirements by Tier" **both** independently state: Rank 1→2 = 5-8 sessions, Rank 2→3 = 8-12 sessions, Rank 3→4 = 12-20 sessions — identical to each other, and noticeably slower than Batch 1 at every tier. With a third data point in, this reads less like an unresolved coin-flip and more like Batch 1 simply using different (faster) numbers than the two later batches, which agree exactly. Still not reconciled — a GM should probably default to the Batch 2/3 numbers given they're the 2-of-3 consensus, but Batch 1's table hasn't been edited to match.

## Downtime activities

Batch 1 gives a generic 4-activity list every background should offer 2-3 of:

1. **Maintain Holdings** — spend time keeping your position
2. **Expand Network** — build new contacts
3. **Generate Income** — use the background to make money
4. **Advance Objectives** — work toward the next rank

Batch 2 refines this into 5 categories keyed to background *type* rather than one generic list — **Combat-Focused** (Gladiator, Monster Hunter, Vampire Hunter, etc.: train, hunt bounties, build reputation, recruit/train juniors), **Social-Focused** (Diplomat, Noble, Entertainer, etc.: attend functions, leverage influence, maintain relationships, expand network), **Knowledge-Focused** (Sage, Archaeologist, Eldritch Scholar, etc.: research, publish, seek sites/knowledge, mentor), **Underworld-Focused** (Criminal, Smuggler, Spy, Charlatan, etc.: run operations, gather intelligence, build networks, maintain cover), and **Spiritual-Focused** (Acolyte, Hermit, Exorcist, Hedge Witch, etc.: perform duties, counsel seekers, maintain sacred spaces, commune with supernatural forces). Not a contradiction — Batch 2's categories are a refinement of Batch 1's generic list, not a replacement.

## DM integration notes

- **Story Hooks**: each background provides built-in quest lines
- **Complications**: Holdings can be threatened, requiring player action to protect them
- **Resources**: higher ranks give players agency to solve problems creatively rather than through combat
- **Power Balance**: income and resources are explicitly framed as significant but not game-breaking
- Background features should complement class features, not replace them — influence and resources are narrative tools, not combat power
- High-rank backgrounds making some challenges trivial is called out as *intended*, not a balance failure
- Design intent: players should feel their background matters as much as their class choice

Batch 3 adds the one genuinely new, concrete rule missing from Batches 1 and 2 — explicit **starting-rank guidance**: most PCs start at Rank 1; experienced characters (5th-10th level) might start at Rank 2; high-level characters (11th+) might start at Rank 3; Rank 4 should always be earned through play, never a starting position. This formalizes what [[Noble]] (Batch 2) only hinted at with its "depending on family standing, you might start at Rank 2 or 3" note — Batch 3 makes that a general rule for every background, not just Noble.

## Setting integration

Explicitly written for **Dór-o Estel**'s themes — the same setting referenced on [[Daldir Lunadreamer]]'s page. [[Acolyte]]'s flavor line ("though the gods are dead") and several dark-fantasy/Grim Hollow-tagged backgrounds (Pit Fighter, Witch Hunter, Plague Doctor, Grave Keeper) confirm this is meant for a grim, post-divine setting rather than a generic high-fantasy one — consistent with [[A World that Watches]]'s tone.

Batch 2 goes further with concrete, named Dór-o Estel factions and places — all first mentioned here, none detailed beyond a name and category, and none yet with dedicated wiki pages (too thin to justify one):

- **Military** ([[Soldier]], [[Knight]]): Veltharyn's Phoenix Knights, Elarion's Border Wardens, Talanthar's Griffon Riders
- **Arcane** ([[Sage]], [[Eldritch Scholar]]): Loom-Cities and Thread research, tying into [[Fracture Whispers]] and [[Unstable Magic Zones]]
- **Dark fantasy** ([[Vampire Hunter]], [[Monster Hunter]], [[Exorcist]]): Sangromancer threats, Corrupted creatures, Eldritch Moon phenomena, DCS trauma/mental attributes
- **Criminal** ([[Criminal]], [[Smuggler]], etc.): urban centers like Kel Varyan, black-market Thread materials, corruption in the Faith of the Bound Pattern (tying into [[Corruption]])

Batch 3 is the first to give these setting names actual mechanical substance rather than just naming them:

- **[[Pattern Keeper]]** confirms the "Faith of the Bound Pattern" is the order that literally maintains the Pattern — the "divine weave that once bound reality" — with concrete rituals for stabilizing damaged Pattern sites and [[Unstable Magic Zones|wild magic zones]].
- **[[Thread Merchant]]** confirms Threads and Glyphs are tradeable spell components with real market value, and that Loom-Cities have "Thread wells" that can be exclusively controlled.
- **[[Fracture Scout]]** is the cartography/survival counterpart to Pattern Keeper's ritual-maintenance approach to the same wild magic zones — the two backgrounds even reference each other's fieldwork at Rank 4.

## Background features tie into Sanity, Focus, Corruption, and Momentum

Batch 2's integration notes state higher ranks should provide: **Sanity bonuses** (success builds confidence — see [[Sanity]]), **Focus benefits** (mastery brings mental discipline — see [[Focus]]), and **Corruption risks** (power at Rank 3-4 may involve dark compromises — see [[Corruption]]). It also calls for **Rally bonuses** (inspiring presence from reputation), **Momentum generation** (fearsome or inspirational reputation affecting combat — see [[Momentum]]), and **Breaking Point protection** (leadership or spiritual strength aiding allies — see [[Breaking Points]]).

**Batch 3 partially fulfills this design intent for Corruption specifically** — its own "Integration with FEARS Systems" section states plainly that religious backgrounds and [[Pattern Keeper]] resist Corruption, while [[Eldritch Scholar]] (Batch 2) and [[Corruption Warden]] risk/engage with it. This isn't just restated intent: [[Inquisitor]] and [[Corruption Warden]] both explicitly grant advantage-on-saves and eventual immunity to Corruption in their actual feature text, and Corruption Warden's Rank 4 even grants the ability to *reduce* someone else's Corruption by 2 tiers once a year — the first background anywhere in this system with a stated numeric Corruption interaction. **Momentum and Breaking Points remain unimplemented** — no background across all 3 batches states a Momentum or Breaking Point mechanic directly, despite both being named in Batch 2's design intent.

## Conversion notes (from standard sources)

Maps each vanilla 5E/Grim Hollow background onto its replacement: Acolyte, Charlatan, Criminal, Entertainer, Folk Hero, Guild Artisan, Sage, and Soldier are directly rank-1-through-4 versions of the PHB backgrounds of the same name. Pauper (Vagabond), Pit Fighter, and Gladiator come from Grim Hollow (Gladiator explicitly distinguished from Pit Fighter — legitimate spectacle vs. underground/illegal fighting). Hermit and Noble (Batch 1's "adaptable but not yet written" list) are now covered by Batch 2. Pauper (Beggar) is noted as "already covered in project files" (not part of this source; not yet located in this vault).

## Still not developed after all 3 batches

**[[Sailor]], [[Urchin]], [[Outlander]]** — all three were on Batch 1's "adaptable but not yet written" list, and none of them appear anywhere in Batch 2 or Batch 3 either. No wiki pages exist for them, and no batch ever circled back to them despite Batch 3 explicitly billing itself as covering "the remaining gaps." Genuinely orphaned at this point — not a documentation gap so much as three PHB backgrounds this system apparently never intends to cover.

## System complete: all 3 batches ingested

`.raw/FEARS Advanced Backgrounds - Batch 1.md`, `Batch 2.md`, and `Batch 3.md` have all been ingested. This page and the 61 background pages above cover the full system as it currently exists in `.raw/`.
