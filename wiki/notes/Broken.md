# Broken

Terminal state reached at **0 [[Sanity]]**: "your soul is no longer entirely yours." The Sanity-track counterpart to [[Burned out]] (Focus's 0-point terminal state).

## Unresolved overlap with Mental State

[[FEARS Character Framework]]'s [[Mental State]] tracker also ends in a stage named "Broken" (Stable → Frayed → Deranged → Broken). Neither source states whether these are the same thing. Two readings are both plausible:

1. Hitting 0 Sanity **is** reaching Mental State's terminal Broken stage — Sanity alone drives the shared tracker.
2. They're independently-named terminal states that happen to share a name — Mental State tracks something broader (a combination of Sanity/Focus/Corruption) while this "Broken" is Sanity-specific.

Not resolved here; flagged on [[Mental State]] and [[FEARS MOC]] as well.
