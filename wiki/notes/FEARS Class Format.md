# FEARS Class Format

The six-part template [[Class_breakdown]] uses uniformly across all 24 classes:

1. **Introduction** — tone and fantasy, usually 2-3 evocative sentences in second person ("You are not trained—you are unleashed.")
2. **What Makes It Unique** — the one differentiating mechanical/thematic hook versus other classes
3. **Signature Abilities (2-3)** — named features, one line each
4. **Gameplay Style** — the table-facing role/archetype label (e.g. "Frontline chaos engine," "Hit-and-run assassin")
5. **Core Loop** — a short arrow-chain of the class's moment-to-moment play pattern (e.g. "Take damage → gain power → hit harder → stay standing → repeat")
6. **Is This You? (Player Prompts)** — 2 yes/no questions aimed at player self-selection, not GM/mechanical content

This is a **presentation/pitch format**, distinct from [[Narrative Identity Framework]]'s six-part *design* framework (Promise/Pressure/Currency/World Reaction/Story Triggers/Reward). The two don't map 1:1 — this format is what a player reads when picking a class; that framework is what a designer fills in to build one. Both could coexist per class without conflict.
