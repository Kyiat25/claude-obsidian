# Mental State

A four-stage tracker layered on top of the raw [[Sanity]]/[[Focus]]/[[Corruption]] numbers from [[A World that Watches]]:

**Stable → Frayed → Deranged → Broken**

[[FEARS Character Framework]] names the tracker but doesn't specify the thresholds that move a character between stages, or which of the three Mental Triad stats (or some combination) drives it.

## Possible partial resolution (from Four Threads of the Mind)

[[Four Threads of the Mind]] states 0 Sanity produces a state also called "[[Broken]]." **Not confirmed as the same thing** — could mean Sanity alone drives this tracker's terminal stage, or could be a separate, coincidentally-named Sanity-specific condition. See [[Broken]] for the full breakdown of both readings. Stable/Frayed/Deranged still have no defined triggers under either reading.
