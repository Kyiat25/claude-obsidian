## Naming collision — resolved by user ruling

"Exploit Weakness" is [[Investigator]]'s signature ability per [[Welcome to FEARS]], but [[Class_breakdown]] gives that exact name to [[Rogue]] instead (renaming Investigator's version "Exploit Vulnerability"). **Resolved**: [[Rogue]]'s own deep-dive never uses "Exploit Weakness" at all (folded into Sneak Attack), so [[Investigator]] keeps sole ownership of both "Exploit Vulnerability" and "Exploit Weakness" as its own studied two-stage progression. Rogue's reactive version of the same fantasy lives inside Sneak Attack, unnamed.

## Shared systems (all 24 classes)

- **Class Reactions** — powered by a shared Proficiency Bonus pool, refreshes on short/long rest
- **Momentum System** — the same [[FEARS Combat System]] mechanic from [[Welcome to FEARS]]
- **Dynamic Combat System (DCS)** — Evasion, Damage Reduction, [[Called Shot|called shots]], location targeting, injury mechanics
- **Hit Dice** — d6 (light casters) to d10 (heavy martial), d8 (hybrids)

## Retired classes — do not use in new content

Shadowshaper, [[Runesmith]], [[Psion]], [[Tamer]], [[Thaumaturge]], [[Shadowbinder]]. See [[Classes_Summary]] for the contradiction note — these five (all but Shadowshaper) have existing wiki pages from [[Welcome to FEARS]], now marked `status: retired` with a contradiction callout rather than deleted.
