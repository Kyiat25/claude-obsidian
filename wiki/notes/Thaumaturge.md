# Thaumaturge

> [!contradiction] Retired per Classes_Summary
> [[Classes_Summary]]'s own note calls this out explicitly by name: the earlier (superseded) version of that summary "included one class — Thaumaturge — that has since been cut from the roster." This page is preserved as a historical record of [[Welcome to FEARS]]'s original pitch, not as current FEARS canon. See [[FEARS Class Roster]] for the active 24-class list.

Forges pacts with entities called **Salutors**, each embodying a fundamental flaw in reality — **Heart, Deed, Mind, and Word** — granting reality-altering abilities. Described as "reality's rebel."

No mechanical detail beyond this one-line pitch is given in the source; the four Salutor types are named but not otherwise defined.
